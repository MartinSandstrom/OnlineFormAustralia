# OnlineFormAustralia

## Usage

### Install

```console
    $ npm install
```

### Build js in dev mode and start dev server

```console
    $ npm start
```
### Start webpack in dev mode and watch files

```console
    $ npm run dev
```

### Deploy to https://martinsandstrom.github.io/OnlineFormAustralia/

```console
    $ npm install -g gh-pages
    $ npm run deploy
```
